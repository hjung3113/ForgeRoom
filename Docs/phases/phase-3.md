# Phase 3 — 확장형

대시보드와 원격 접속, 분산·병렬화.

## 항목

- 데스크탑 앱 UI (대시보드, 칸반, 파이프라인 그래프, 활동 로그)
- Tailscale 통합 (앱 원격 접속)
- 한 task 내 병렬 sub-task
- 커스텀 CLI agent 직접 등록 (OpenClaw 외부)
- 조건 분기 `when`
- 외부 `tool` 호출 (LSP, formatter, custom scripts)
- 워크플로우 import / include
- 동적 step 생성 (런타임 step 삽입/삭제)
- 작업 재시도 큐, 실패 자동 회복
- 다중 머신 orchestrator (cluster) — 검토 단계

## 미확정 항목

- 데스크탑 앱 기술 스택 (Tauri vs Electron)
- 클러스터링 필요성 (단일 머신으로 충분한지)
- 외부 tool 호출 보안 모델
